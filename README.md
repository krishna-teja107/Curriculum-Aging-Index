# 📊 Curriculum Aging Index

**Curriculum Aging Index (CAI)** is an AI-powered platform that quantitatively measures how outdated or industry-relevant an academic curriculum is.  
It bridges the long-standing gap between education and employability by auditing syllabi against real-time industry skill demands and generating actionable learning roadmaps.

---

## 🧩 Domain
**Education and Employment**

---

## 🎯 Problem Statement

Academic curricula often lag behind rapidly evolving industry requirements. As a result, many students graduate without the in-demand skills required in today’s job market, creating a significant employability gap.

There is currently no standardized, data-driven way to measure how outdated a curriculum is or to continuously audit it against real-world industry needs.

---

## 💡 Proposed Solution

The **Curriculum Aging Index** measures curriculum relevance by comparing syllabus content with current industry skills and update frequency.

It:
- Identifies obsolete topics
- Detects missing in-demand skills
- Highlights modern and relevant content
- Generates personalized learning paths to bridge gaps

The system is designed as a **living audit mechanism** that dynamically updates as industry skills evolve.

---

## 🚀 Key Features

### 1️⃣ AI-Powered Curriculum Aging Index
- Computes a **0–100 Aging Index**
- Clearly indicates how outdated or industry-ready a curriculum is

### 2️⃣ Intelligent Gap & Relevance Analysis
- Detects obsolete topics and missing skills
- Highlights modern strengths
- Visual analytics for quick understanding

### 3️⃣ Career-Aligned Skill Bridging
- AI-generated learning roadmaps
- Practical challenges and assessments
- Improves role-specific readiness

### 4️⃣ Verified Skill Progress & Persistence
- Skill validation through quizzes
- Tracks learning progress
- Dynamically updates the Aging Index over time

---

## 🛠 Google Technologies Used

### 🔹 Gemini AI (Core Intelligence Engine)
- Uses **Gemini 3 Flash Preview**
- Performs syllabus analysis, skill gap detection, roadmap generation, and assessments
- Produces structured JSON outputs for reliable UI rendering

### 🔹 Multimodal Curriculum Understanding
- Directly analyzes uploaded PDF syllabi using Gemini’s document vision capabilities

### 🔹 AI-Generated Visual Blueprints
- Uses **Gemini 2.5 Flash Image**
- Converts analysis results into architecture and process flow diagrams

> Built entirely on Google’s AI and web ecosystem for speed, accuracy, and scalability.

---

## 🔄 Process Flow

1. User authentication
2. Selection of academic stream, domain, and target role
3. Upload of syllabus document
4. AI analysis against current industry benchmarks
5. Identification of obsolete content, skill gaps, and modern skills
6. Computation of Curriculum Aging Index
7. Generation of a personalized learning roadmap and bridge strategy
